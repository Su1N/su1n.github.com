Elasticsearch权威指南笔记

阅读计划：2019-11-11 —— 2019-11-15

Introduction

关键字翻译约定

  • index 索引
  • type 类型
  • token 表征
  • filter 过滤器
  • analyser 分析器

入门

描述定位

  • 分布式的实时文件存储，每个字段都被索引并可被搜索
  • 分布式的实时分析搜索引擎
  • 可以扩展到上百台服务器，处理PB级结构化或非结构化数据

集群和节点

节点 node：是一个运行着的Elasticsearch实例
集群(cluster)：是一组具有相同 cluster.name 的节点集
